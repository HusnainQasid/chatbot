# 🤖 FAQ Chatbot (NLP Based)

## 📌 Overview
This project is a simple FAQ Chatbot built using Python, NLP (NLTK), and Flask.

It matches user queries with stored FAQs using **TF-IDF + Cosine Similarity**.

---

## 🚀 Features
- NLP preprocessing (tokenization, stopword removal)
- TF-IDF vectorization
- Cosine similarity matching
- REST API using Flask
- Simple chatbot response system

---

## 🛠️ Technologies Used
- Python
- NLTK
- Scikit-learn
- Flask
